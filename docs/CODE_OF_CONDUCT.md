# DataShield Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inspiring community for all. We expect all participants to adhere to this Code of Conduct.

## Expected Behavior

- Be respectful and inclusive
- Be patient and understanding
- Focus on constructive feedback
- Respect differing opinions and experiences

## Unacceptable Behavior

- Harassment or discrimination
- Offensive comments or language
- Personal attacks
- Unwelcome advances or attention

## Reporting Issues

If you experience or witness unacceptable behavior, please contact the maintainers.

## Enforcement

Violations of this code of conduct may result in temporary or permanent ban from the community.

This Code of Conduct is adapted from the Contributor Covenant.
